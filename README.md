Flutter Authentication UI

This project is a simple Flutter authentication UI.
It includes a Login page and a Create Account page.
A reusable logo widget is used across screens.
Built using Flutter and Material Design.
The UI is clean, responsive, and easy to customize.
Designed for mobile applications.
Authentication logic can be added later.
Suitable for student and demo projects.
Easy to extend and maintain.
